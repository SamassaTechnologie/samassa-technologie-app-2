Activer la base de données (Realtime Database)
À faire
Dans votre projet Firebase :
1Dans le menu gauche, cliquez sur "Realtime Database" (sous "Build")
2Cliquez "Créer une base de données"
3Choisissez l'emplacement : europe-west1 (Belgium) → Cliquez "Suivant"
4Choisissez "Démarrer en mode test" → Cliquez "Activer"
5Allez dans l'onglet "Règles" (Rules), remplacez tout le contenu par :
{ "rules": { ".read": true, ".write": true } }
6Cliquez "Publier" pour enregistrer les règles
💡 Notez l'URL de votre base de données Elle ressemble à : https://samassa-kayes-default-rtdb.europe-west1.firebasedatabase.app

❓ Questions fréquentes

Est-ce que c'est gratuit ?

Oui, le plan gratuit Firebase (Spark) permet jusqu'à 1 Go de données et 10 Go de transfert par mois. C'est largement suffisant pour cette application.

Mes données sont-elles sécurisées ?

Les données sont stockées sur les serveurs de Google (Firebase), protégées par HTTPS. Seules les personnes avec l'URL de votre base peuvent accéder aux données.

Que se passe-t-il si je n'ai pas Internet ?

L'application fonctionne normalement hors-ligne. Dès que la connexion revient, les données se synchronisent automatiquement.

J'ai une erreur "Permission denied"

Vérifiez que vous avez bien publié les règles de sécurité à l'étape 3. Les règles doivent contenir ".read": true, ".write": true.

J'ai une erreur "Invalid API key"

Vérifiez que vous avez bien copié la valeur apiKey complète sans espaces dans le fichier firebase-config.js.
