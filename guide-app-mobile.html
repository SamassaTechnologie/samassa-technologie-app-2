⚡ Méthode Capacitor — Guide complet
📋 Ce dont vous avez besoin Un ordinateur Windows, Mac ou Linux avec une connexion Internet. Le processus prend environ 1 à 2 heures la première fois.
1
Installer Node.js
Environnement nécessaire pour Capacitor
À faire

Node.js est le moteur qui permet d'utiliser Capacitor sur votre ordinateur.

📥 TéléchargementAllez sur nodejs.org et téléchargez la version LTS (20.x). Installez-la normalement.

Vérifiez l'installation en ouvrant le terminal (cmd sur Windows) et tapez :

Terminal / Invite de commandes
node --version
Si vous voyez v20.x.x ou similaire, Node.js est installé ✓
2
Installer Android Studio
Nécessaire pour compiler l'APK Android
À faire
📥 TéléchargementAllez sur developer.android.com/studio et téléchargez Android Studio. Taille : ~1 Go.

Lors de l'installation, acceptez tous les paramètres par défaut. Android Studio va installer :

  • Android SDK (outils de compilation)
  • JDK 17 (Java nécessaire)
  • Émulateur Android (pour tester sans téléphone)
⏱️ Temps d'installationAndroid Studio prend 20-30 minutes à s'installer et configurer la première fois.
3
Installer les dépendances Capacitor
Dans le dossier de votre projet
À faire

Ouvrez un terminal dans le dossier de votre projet Samassa (là où se trouvent vos fichiers HTML) et tapez ces commandes une par une :

Commande 1 — Installer Capacitor
npm install
Commande 2 — Initialiser Capacitor
npx cap init
💡 Le fichier capacitor.config.json est déjà inclus dans votre ZIPIl est déjà configuré avec le nom "Samassa Tech Pro" et l'identifiant app correct.
Commande 3 — Ajouter Android
npx cap add android
Un dossier android/ sera créé dans votre projet ✓
4
Synchroniser et ouvrir Android Studio
Copier les fichiers web dans le projet Android
À faire
Synchroniser les fichiers web → Android
npx cap sync android
Ouvrir dans Android Studio
npx cap open android
📱 Android Studio s'ouvre avec votre projetAttendez qu'il finisse de synchroniser Gradle (barre de progression en bas).
5
Ajouter l'icône et l'écran de démarrage
Personnaliser l'apparence de l'app
À faire

Dans Android Studio, allez dans app → res et remplacez les icônes par votre logo :

🎨 Plus simple : utiliser l'outil Image Asset Dans Android Studio : Clic droit sur res → New → Image Asset → importez votre icon-512x512.png → il génère toutes les tailles automatiquement.
Ou avec npx (recommandé)
npx @capacitor/assets generate --iconBackgroundColor '#0D1B2A' --splashBackgroundColor '#0D1B2A'
6
Générer l'APK Android
Créer le fichier d'installation
À faire

Dans Android Studio :

  • Menu Build → Build Bundle(s) / APK(s) → Build APK(s)
  • Attendez la compilation (2-5 minutes)
  • Cliquez "locate" dans la notification en bas
  • L'APK se trouve dans android/app/build/outputs/apk/debug/
✅ Vous avez votre APK ! Envoyez ce fichier .apk par WhatsApp ou câble USB à votre téléphone et installez-le.
⚠️ Activer "Sources inconnues" sur Android Paramètres → Sécurité → Installer des apps inconnues → Autoriser pour votre gestionnaire de fichiers.
7
iPhone / iPad (iOS) — Optionnel
Nécessite un Mac avec Xcode
Optionnel
⚠️ Requis : Mac avec macOSLa compilation iOS nécessite obligatoirement un Mac avec Xcode. Impossible sur Windows.
Sur Mac uniquement
npx cap add ios
npx cap sync ios
npx cap open ios
💡 Alternative sans MacUtilisez Expo EAS Build (service cloud payant) ou partagez votre code avec quelqu'un qui a un Mac.
8
Mettre à jour l'application
Après chaque modification du site
Automatique

Quand vous modifiez vos fichiers HTML/CSS/JS, lancez simplement :

Synchroniser et reconstruire
npx cap sync
npx cap open android
Puis Build → Build APK(s) dans Android Studio.
🌐 PWA — Déjà activée sur votre site
✅ Votre application est déjà installable en PWA ! Aucune installation d'outils nécessaire. Faites simplement ces étapes sur chaque téléphone.
1
Sur Android (Chrome)
Simple
  • Ouvrez votre site dans Google Chrome sur Android
  • Appuyez sur les ⋮ 3 points en haut à droite
  • Sélectionnez "Ajouter à l'écran d'accueil"

2
Sur iPhone / iPad (Safari)
Simple
  • Ouvrez votre site dans Safari sur iPhone
  • Appuyez sur le bouton Partager ⬆ (barre du bas)
  • Descendez et appuyez sur "Sur l'écran d'accueil"
  • Appuyez sur "Ajouter" en haut à droite
L'icône de votre app apparaît sur l'écran d'accueil ✓

❓ Questions fréquentes
Le fichier APK peut-il être partagé par WhatsApp ? ▼
Oui ! Envoyez directement le fichier .apk par WhatsApp ou par câble USB. Vos employés l'installent en activant "Sources inconnues" dans les paramètres Android.
La synchronisation Firebase fonctionne-t-elle dans l'APK ? ▼
Oui, exactement comme dans le navigateur. Le fichier firebase-config.js est inclus dans l'APK et la sync fonctionne identiquement.
Faut-il publier sur le Play Store ? ▼
Non. Vous pouvez distribuer votre APK directement par WhatsApp ou par email. Le Play Store est optionnel (et nécessite 25$ de frais).
Le mot de passe fonctionne-t-il dans l'app ? ▼
Oui ! Le système auth.js fonctionne identiquement dans l'APK. Au lancement, la page login.html s'affiche automatiquement.
Peut-on avoir plusieurs appareils avec la même app ? ▼
Oui. Envoyez le même APK à tous vos appareils. Avec le même firebase-config.js, tous les appareils voient les mêmes données en temps réel.
